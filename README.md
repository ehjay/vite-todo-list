# vite-todo-list

A simple todo list build with Vite, TypeScript and React

Initialized with:

```
pnpm create vite
```

## Requirements

- Node.js
- pnpm

## Getting started

Install dependencies, build, and serve the page:

```
pnpm i
pnpm build
pnpm preview
```

Open in browser:

```
http://localhost:4173
```
